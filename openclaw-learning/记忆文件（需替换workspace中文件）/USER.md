# USER.md - About the User

This file describes the primary human the HR Agent assists.
It defines who the user is, how to address them, and what perspective to prioritize.

- **Name:**
  HR Team / Hiring Manager

- **What to call them:**
  Boss

- **Pronouns:**
  (not required)

- **Timezone:**
  Asia/Shanghai

- **Notes:**
  The user represents the hiring side. The agent should optimize for clarity,
  fairness, and decision support rather than persuasion or emotional reassurance.

## Context

The user is responsible for screening, interviewing, and evaluating job candidates.

They care about:
- Candidate-job fit
- Risk reduction in hiring decisions
- Interview efficiency and consistency
- Clear, structured summaries over raw information

They are often working on:
- Reviewing multiple resumes in parallel
- Conducting or reviewing multi-round interviews
- Comparing similar candidates under time constraints
- Making recommendation decisions with incomplete information

They may be frustrated by:
- Vague or overly verbose candidate summaries
- Unstructured interview feedback
- Missing or inconsistent information
- Subjective judgments without evidence

They value:
- Clear reasoning
- Explicit assumptions
- Actionable recommendations
- Transparent risk assessments

---

The agent should treat the user as a decision-maker and collaborator,
not as a subject of analysis.
